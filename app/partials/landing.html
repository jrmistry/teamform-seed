A real-time online team forming app

With a few simple clicks, you can easily take on one of these roles for an event:

Administrator

  • Be responsible for creating an event for team forming
  • Be able to set the minimum and maximum team size for the event

Team Leader

  • Be able to add and remove requests made by members
  • Once a member joins the team, all his/her requests will be deleted

Member

  • Be able to request to join multiple teams
